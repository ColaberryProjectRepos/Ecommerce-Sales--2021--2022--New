Creating and Formatting the Line and Stacked Column Chart for Sales per Yr/Qrt/Mth. 18

  • Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Column Char


Build Visual Tab:

  • X axis = Year, Year Quarter, Month Year
  • Column y-axis = Sales per Month
  • Line y-axis = Sales MOM%, GPM, GPM MOM%, Sales YOY%,
  • Tooltips = Sales MOM%, GPM, GPM MOM%, YOY Sales %
  • Sort axis by Month Year and ascending order


Format Visual Tab:

  • General: Properties, Size, Height=311, Width=538, Position, Horizontal=231, Vertical=108,  Title=On, Text Sales Per Yr/Qrt/Mth, Font=Segoe UI Semi Bold (14), Text color #FFFFFF, Background color=#118DFF, Horizontal alignment=Center
  •  Effects: Background=On, Background color=#e6e6e6 
  •  Visual: X-axis: Values: Font Segoe Semi Bold (8), Font color #6c6966    Title = Off Y-axis: Values: Font=Segoe UI Bold (9), Font color #6c6966, Display Units=Millions  Title = Off
  •  Secondary Y-axis: Range: Align Zeros  Off
  •  Values: Font=Segoe UI Semi Bold (9), Font color=#6c6966, Display Units=Auto
  •  Title = Off Gridlines:  Off
  • Lines: Colors: Sales MOM% #12239E, GPM #E66C37, GPM MOM% #4A2D75, Sales  
  • YOY% E645AB, Shade area=On
  • Markers Off
  • Columns: Colors #188DFF
  • Data labels: Sales per Month ON, Sales MOM% Off
  • GMP Off , GPM MOM% Off, Sales YOY% Off
  • Sales per Mth: Position(Column)= Auto, Orientation (Column)= Vertical
  • Values: Font=Segoe UI  Semi Bold(9), Color=#6C6966, Value decimal places= Auto
  • Background = On
