Creating Calendar Table



  • Go to Modeling Tab click on new table.
  • Type the following DAX formula on DAX formula bar, click on the check mark and save.


Calendar = ADDCOLUMNS (

CALENDAR (Date(Year(Min('ecommerce_data (3)'[order_date])),

Month(Min('ecommerce_data (3)'[order_date])),

DAY(Min('ecommerce_data (3)'[order_date]))),

Date(Year(Max('ecommerce_data (3)'[order_date])),

Month(Max('ecommerce_data (3)'[order_date])),

DAY(Max('ecommerce_data (3)'[order_date])))),

"Month number", FORMAT ([Date], "MM"),

"Month Year", FORMAT ([Date], "MMMYYYY"),

"Month", FORMAT ([Date], "mmmm"),

"Week Number", "Week-"& WEEKNUM([Date]),

"Week", WEEKNUM ([Date]),

"Day Of Month", Month([DATE]) &"-"& FORMAT ( [Date], "dd" ),

"Year Quarter", FORMAT ([Date], "YYYY") & "/Q" & FORMAT ([Date], "Q"),

"Year", FORMAT ([Date], "YYYY")

)

Creating Refresh Date

  • On the home page under Get Data select Blank Query and Rename it to Refresh Date
  • On the Formula window enter this Formula Datetime.LocalNow()
  • A date time will appear
  • The on the top left change it to a table
  • Rename the Column to Last Refresh and Save
