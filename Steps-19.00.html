Creating Location Map and Legend 19

                   

  • Select Shape Map, on Location add Customer State , on Color Saturation add Sales per Month.
  • In the Formatting Tab-->Visual-->Fill colors: Gradient On, Minimum #DFFFFF, Center #90D1FF, Maximum #118DFF
  • Take screenshot of the Gradient and save it in your device so that you can use it later as a Legend for the Map.
  • In the Formatting Tab-General-Properties-Size: Height 309, Width 517, Position, Horizontal 762, Vertical 88. Title On Text Sales by State.
  • To create the Map Legend. In the report view, Select Insert ribbon>Text box >Type  ‘Low and High’, Highlight the Text and adjust Size 10, Font Segoe Semi (Bold), color   Black, background Off.
  • In the Formatting Tab-General--Properties--Size: Height 36, Width 256, Position, Horizontal 893, Vertical 357.
  • In the report view, Select Insert ribbon Image   and select the Gradient Image you created earlier at no. 3.
  • In the Formatting Tab-General- Properties- Size: Height 45, Width 155, Position: Horizontal 942, Vertical 352, Background: Off.
