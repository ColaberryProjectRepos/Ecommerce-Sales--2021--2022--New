Inserting the Colaberry Logo, The dashboard title and Project Logo 13

 



Use this Specifications to format the colaberry Logo

  • On the Home page of the powerbi ribbon select Insert then click on the Image button, select the saved colaberry logo
  • Image: Style-Scaling: Normal
  • Properties Size Height: 98, Width:233
  •  Position Horizontal:0, Vertical : 1
  • Padding 5px all around
  • Background ON: White

For the Title Select Insert Text Box and use below Formatting

  • General: Properties Size Height: 100, Width: 779
  •  Position Height:250, Vertical:0
  • Padding  5px  all around
  • Title ON: TEXT: ECOMMERCE SALES (2021-2022) Font: Segoe UI Semi bold, Font size 32 Text Color #118DFF, Horizontal
  • Alignment Center
  • Background ON: White

For the project logo go to insert Image and select the Image. Format it as below

  • Image-Style-Scaling: Fill
  • General: Properties Height: 92, Width: 213
  • Position Height: 1046 Vertical: 4
  • Padding 5px all round
  • Background ON: White
