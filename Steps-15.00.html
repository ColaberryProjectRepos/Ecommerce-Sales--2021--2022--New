Adding KPIS 15

  •  To create the KPIs , drag and drop the card visual
  • Drag and drop Total Sales field to the card visual
  • Use the following format
  • Visual: Callout value Font Segoe UI Size 18 Color #000000
  • Display units auto, Value decimal auto General tab: Properties size Height 105 Width 214, Position-  Horizontal 0 Vertical 234, padding 0x on top 5px on sides
  • Title ON Title Total Sales Font Segoe UI 14 Color #FFFFFF Background #118DFF Horizontal Alignment center

  Make 2 copies of the card visual for the for the two remaining KPIs, main KPIs Profit and YOY Sales %

  • The first copy of the card visual change the field to Profit
  • Visual: Callout value Font Segoe UI Size 18 Color #000000, Display units auto, Value decimal auto
  • General tab: KPI Position  Horizontal  0 Vertical 357

 The second copy of the card visual change the field to YOY Sales %

  • Properties size Height 105 Width 214 Visual: Callout value Font Segoe UI Size 18, Color #ED45AB
  • Display units auto, Value decimal auto
  • Position  Horizontal 0 Vertical 508
