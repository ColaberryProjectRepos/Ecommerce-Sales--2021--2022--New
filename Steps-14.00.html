Creating Slicer 14

Open Visualization pane Select Slicer visual

  •  Drag the visual to the dashboard.
  • Click on Date table and drag Year column from the Date Hierarchy then place it inside the field.
  • General: Properties, Height  112, and Width 232, Position, Horizontal 12, Vertical 89
  • Title On, Text Year, Font Segoe UI Bold (14), Text color #FFFFFF, Background color #118DFF, Horizontal alignment Center
  • Effects, Background On, Background color #E6E6E6
  • Visual: Values, Font Segoe UI Bold(8), Font color #000000 Slider On
